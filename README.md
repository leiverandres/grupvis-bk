## Dependencies

## Start with the project
* Create a virtual enviroment

```bash

```

* Install the modules
```bash
pip install -r top-requirements.txt
```

* Download the data and save it in `datasets/` folder

* Run jupyter notebook

```bash
jupyter notebook
```
